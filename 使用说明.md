# 使用说明

## 📦 openclaw-installer-pro-v1.0.0.zip

本压缩包包含 OpenClaw Installer Pro 的完整文件，支持多种安装方式。

## 🚀 快速开始

### Linux 用户

**方式一：直接运行可执行文件（推荐）**

```bash
# 解压
unzip openclaw-installer-pro-v1.0.0.zip

# 赋予执行权限
chmod +x openclaw-installer

# 直接运行
./openclaw-installer
```

**方式二：使用安装脚本**

```bash
# 解压
unzip openclaw-installer-pro-v1.0.0.zip

# 运行安装脚本
./install.sh
```

**方式三：从源码运行**

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python src/main.py
```

### macOS 用户

```bash
# 解压
unzip openclaw-installer-pro-v1.0.0.zip

# 运行安装脚本
./install.sh

# 或从源码运行
python3 src/main.py
```

### Windows 用户

**重要说明**：当前 zip 包中的可执行文件仅支持 Linux 系统。Windows 用户有以下选择：

**方式一：使用安装脚本（需要安装 Python）**

1. 下载并安装 Python 3.8+: https://www.python.org/downloads/
2. 解压 zip 文件
3. 双击运行 `install.bat`

**方式二：从源码运行（需要 Python）**

```cmd
pip install -r requirements.txt
python src/main.py
```

**方式三：编译 Windows 可执行文件（需要 Windows 环境）**

在 Windows 系统上执行：

```cmd
pip install pyinstaller
pyinstaller --onefile --windowed --name "OpenClaw-Installer-Pro" src/main.py
```

生成的 `OpenClaw-Installer-Pro.exe` 可以在没有 Python 的系统上直接运行。

## 📝 系统要求

### Linux/Mac
- Linux: Ubuntu 20.04+, Debian 11+, 或其他主流发行版
- macOS: 10.15 或更高版本
- 内存: 至少 2GB RAM
- 磁盘空间: 至少 2GB 可用空间

### Windows
- Windows 10 或更高版本
- Python 3.8 或更高版本（除非使用预编译的 .exe）
- 内存: 至少 2GB RAM
- 磁盘空间: 至少 2GB 可用空间

## 🔧 配置说明

启动程序后，需要配置以下信息：

1. **模型地址**：大模型 API 的完整 URL
   - 智谱 AI: `https://open.bigmodel.cn/api/paas/v4/chat/completions`
   - 百度文心: `https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop/chat/completions_pro`
   - 阿里通义: `https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation`

2. **模型 ID**：要使用的模型名称
   - 智谱: `glm-4`, `glm-4-flash`, `glm-3-turbo`
   - 百度: `ernie-bot-4`, `ernie-bot-turbo`
   - 阿里: `qwen-turbo`, `qwen-plus`

3. **API Key**：你的访问密钥

4. **代理（可选）**：HTTP 代理地址
   - 格式: `http://192.168.48.1:7890`

## 💡 获取 API Key

### 智谱 AI
1. 访问 https://open.bigmodel.cn/
2. 注册并登录
3. 进入控制台 → API Keys
4. 创建新的 API Key

### 百度文心
1. 访问 https://cloud.baidu.com/
2. 注册并登录
3. 进入 AI 开发平台 → 创建应用
4. 获取 API Key 和 Secret Key

### 阿里通义
1. 访问 https://dashscope.aliyun.com/
2. 注册并登录
3. 进入控制台 → API-KEY 管理
4. 创建 API Key

## ❓ 常见问题

### Q: Linux 可执行文件无法运行？

A: 检查执行权限：
```bash
chmod +x openclaw-installer
```

### Q: Windows 怎么用？

A: Windows 用户目前需要安装 Python 后运行，或在 Windows 环境编译 .exe 文件。

### Q: 下载速度慢？

A: 在"代理设置"中配置代理地址，如 `http://192.168.48.1:7890`

### Q: 支持哪些模型？

A: 支持所有兼容 OpenAI API 格式的大模型。
