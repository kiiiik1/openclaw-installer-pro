# OpenClaw Installer Pro v1.0.0

## 快速安装

### Windows 用户

1. 解压 `openclaw-installer-pro-v1.0.0.zip`
2. 双击运行 `install.bat`
3. 等待依赖安装完成
4. 程序会自动启动

### Linux/Mac 用户

```bash
# 解压
unzip openclaw-installer-pro-v1.0.0.zip

# 运行安装脚本
cd openclaw-installer-pro-v1.0.0
./install.sh
```

## 手动安装

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python src/main.py
```

## 系统要求

- Python 3.8 或更高版本
- 网络连接（可选配置代理）
- 至少 2GB 可用内存

## 配置说明

启动程序后，填写以下信息：

1. **模型地址**：API 的完整 URL
   - 智谱 AI: `https://open.bigmodel.cn/api/paas/v4/chat/completions`
   - 百度文心: `https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop/chat/completions_pro`

2. **模型 ID**：要使用的模型名称
   - 智谱: `glm-4`
   - 百度: `ernie-bot-4`

3. **API Key**：你的 API 密钥

4. **代理（可选）**：`http://192.168.48.1:7890`

## 获取 API Key

### 智谱 AI

1. 访问 https://open.bigmodel.cn/
2. 注册并登录
3. 进入控制台，创建 API Key

### 百度文心

1. 访问 https://cloud.baidu.com/
2. 注册并登录
3. 进入 AI 开发平台，创建应用

## 常见问题

### Q: 下载速度慢怎么办？

A: 配置代理，在"代理设置"中填写代理地址

### Q: 安装失败？

A: 检查 Python 版本是否 >= 3.8，确保网络连接正常

### Q: 如何卸载？

A: 直接删除安装目录即可
