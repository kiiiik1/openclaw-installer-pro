## 产品简介

OpenClaw Installer Pro 是一款专为国内用户打造的 OpenClaw 一键部署工具。通过图形化界面，用户只需配置3个参数即可完成 OpenClaw 的安装、配置和启动。

### 什么是 OpenClaw？

OpenClaw 是一个开源的 AI 助手框架，支持多种聊天平台（QQ、微信、Telegram 等），通过技能系统实现各种功能扩展。

### 为什么选择本产品？

- ✅ **极简操作** - 无需命令行，全程图形化
- ✅ **快速部署** - 3步完成安装，5分钟内启动
- ✅ **预置技能** - 内置常用技能，开箱即用
- ✅ **国内优化** - 支持代理，适配国内网络环境
- ✅ **持续更新** - 跟随 OpenClaw 最新版本更新

## 安装指南

### Windows 用户

1. 下载安装包
2. 双击运行 `openclaw-installer-pro.exe`
3. 按照界面提示操作

### Linux 用户

#### 方法一：使用 pip 安装

```bash
pip install openclaw-installer-pro
openclaw-installer
```

#### 方法二：从源码安装

```bash
# 克隆仓库
git clone https://github.com/kiiiik1/openclaw-installer-pro.git
cd openclaw-installer-pro

# 安装依赖
pip install -r requirements.txt

# 运行程序
python src/main.py
```

### 系统要求

- **操作系统**：Windows 10/11 或 Linux (Ubuntu 20.04+, Debian 11+)
- **Python**：3.8 或更高版本（Linux）
- **内存**：至少 2GB RAM
- **磁盘空间**：至少 2GB 可用空间
- **网络**：需要互联网连接（可选配置代理）

## 使用教程

### 第一步：准备 API

在使用 OpenClaw 之前，你需要一个大模型 API。

#### 推荐的 API 提供商

| 提供商 | 模型地址 | 获取方式 |
|--------|----------|----------|
| 智谱 AI | https://open.bigmodel.cn/api/paas/v4/chat/completions | https://open.bigmodel.cn/ |
| 百度文心 | https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop/chat/... | https://cloud.baidu.com/ |
| 通义千问 | https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation | https://dashscope.aliyun.com/ |

以智谱 AI 为例：
1. 访问 https://open.bigmodel.cn/
2. 注册并登录
3. 进入控制台，创建 API Key
4. 记录 API Key

### 第二步：配置安装器

1. 启动 OpenClaw Installer Pro
2. 填写配置信息：
   - **模型地址**：`https://open.bigmodel.cn/api/paas/v4/chat/completions`
   - **模型 ID**：`glm-4`
   - **API Key**：你申请的 API Key
3. 选择安装路径（默认为 `~/openclaw`）
4. 如需要，配置代理（如 `http://192.168.48.1:7890`）

### 第三步：开始安装

点击"开始安装"按钮，等待安装完成。

安装过程包括：
- 下载 OpenClaw
- 配置模型和 API
- 安装内置技能
- 创建启动脚本

### 第四步：启动 OpenClaw

安装完成后，有两种启动方式：

#### 图形界面启动
- **Windows**：双击安装目录下的 `start.bat`
- **Linux/Mac**：运行 `./start.sh`

#### 命令行启动
```bash
cd ~/openclaw
openclaw start
```

### 第五步：访问 OpenClaw

打开浏览器，访问 `http://localhost:3030`，即可进入 OpenClaw 网页界面。

### 第六步：配置聊天平台

在 OpenClaw 网页界面中：
1. 进入"设置" → "通道"
2. 添加 QQ、微信等聊天平台
3. 扫码或填写配置信息
4. 保存并启用

完成！现在你可以通过聊天平台与 AI 助手对话了。

## 常见问题

### 1. 下载速度慢怎么办？

**解决方案**：
- 配置代理：在安装器的"代理设置"中填写代理地址
- 使用 Gitee 源：程序会自动尝试从 Gitee 下载（国内更快）

### 2. 安装失败怎么办？

**常见原因及解决**：
- **网络问题**：检查网络连接，或配置代理
- **权限不足**：以管理员身份运行（Windows）或使用 sudo（Linux）
- **磁盘空间不足**：确保至少有 2GB 可用空间

### 3. 支持哪些模型？

支持所有兼容 OpenAI API 格式的大模型，包括：
- 智谱 GLM 系列
- 百度文心系列
- 阿里通义系列
- 以及其他 OpenAI 格式兼容的模型

### 4. 如何卸载？

直接删除安装目录即可：
```bash
rm -rf ~/openclaw
```

### 5. 可以同时配置多个模型吗？

可以！在 OpenClaw 的配置文件中，可以添加多个模型，并在对话时切换使用。

### 6. 如何更新 OpenClaw？

运行以下命令：
```bash
openclaw update
```

或重新运行安装器，程序会自动更新到最新版本。

### 7. 技能如何使用？

技能在 OpenClaw 启动后自动加载。通过聊天平台发送消息时，AI 会根据内容自动调用相应技能。

你也可以通过网页界面查看和管理技能。

### 8. Windows 上无法启动？

**检查项**：
1. 确保 Windows 10 或更高版本
2. 检查是否有安全软件拦截
3. 以管理员身份运行启动脚本
4. 查看 `start.log` 日志文件
